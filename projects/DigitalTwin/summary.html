Overview

3 different versions of DigitalTwin of CO Office.

Version 1: The Explorer

This version allows you to explore the CO office along with CO projects that are displayed within the space. You can load each project using a dropdown menu and simply walk into it.

Version 2: The Time Machine

This version lets you explore different stages of the build process. You can overlay information from past phases of the office directly onto the 3D model.

Version 3: Live Data

This version includes real-time data, such as temperature. It also allows you to simulate virtual versions of activities that happen in the building and observe them.
